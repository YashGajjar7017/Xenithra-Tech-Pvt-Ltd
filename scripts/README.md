Windows helper scripts:

- `run-dev.bat` — starts `npm run dev` for local development (HMR).
- `preview.bat` — runs `npm run build` and then `npm start` to preview a production build.

Usage:

1. Open PowerShell or cmd in project root.
2. Run `scripts\run-dev.bat` to start development.
3. Run `scripts\preview.bat` to build + preview.
